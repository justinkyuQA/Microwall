# MicroWall

A simplified firewall analyzer.

Features

- Read firewall rule files
- Count allow/block rules
- Display rules
- Educational CLI

---

## Overview

Minimal educational firewall utility demonstrating basic filtering and traffic control concepts.

## Features

- Lightweight command-line utility
- Educational focus
- Cross-platform Python
- Fast execution
- Minimal dependencies

## Educational Concepts

- Firewalls
- Packet filtering
- Network security
- Access control

## Part of the Micro Toolkit

This project is one member of the Micro Toolkit, a collection of focused educational networking and cybersecurity utilities. Each project is designed to teach one concept well while remaining simple enough to understand, modify, and extend.
